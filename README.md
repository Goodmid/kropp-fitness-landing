# Kropp Fitness Landing Page / Целевая страница Kropp Fitness

## 🇬🇧 English

A clean and responsive fitness landing page built with pure HTML and CSS.

📖 About

Kropp Fitness is a static landing page focused on responsive layout, typography, and clean UI design.
The project was implemented based on a Figma design, ensuring pixel-perfect accuracy and attention to detail.

🚀 Features:

📱 Fully responsive design
🎨 Clean and modern UI
🧱 Structured layout and reusable sections
⚡ Fast loading (no JavaScript)
🎯 Pixel-perfect implementation from Figma

🛠 Tech Stack:
HTML5
CSS3

📦 Installation & Setup

git clone https://github.com/YOUR_USERNAME/kropp-fitness.git
cd kropp-fitness

Open in browser: index.html
